1 ОПИС АРХІТЕКТУРИ IOT ЧАСТИНИ

Для розробки IoT було прийнято рішення використовувати мову програмування Java для написання заміни системи IoT, але не дивлячись на те, що система буде заміною системи IoT, вона буде виконувати свої функції як необхідно і створювати випадкові значення, що будуть оброблятися та відправлятися на сервер, тобто система буде мати приближений до справжнього IoT пристрою функціонал. Для взаємодії між IoT сервісом та серверною частиною буде використано hyper-text tranfer protocol (HTTP) в поєднані з REST. Сам IoT сервіс буде написано на мові програмування Java з використанням фреймворку Spring Boot.
